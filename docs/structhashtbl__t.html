◆ cap

size_t hashtbl_t::cap

Total number of elements that can be stored before expanding the table.

Recomputed only when the table actually expands. In particular, inserting multiple elements with different load factors in the function table will not cause a re-hash (hashtbl_ft::load_factor, hash_insert).
