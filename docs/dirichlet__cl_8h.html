DESCRIPTION

OpenCL implementations of functions in dirichlet.h and dirichlet_powerful.h Generally, these functions require an OpenCL manager (nut_ClMgr) and if mgr->flags does not have die or verbose set, these functions will usually return 0 on error so if 0 is returned and neither die nor verbose is set, check mgr->err before relying on the result (eg by calling nut_cl_check_err).
